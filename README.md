#  React Home Page – Counter & Theme App

A simple and clean React application demonstrating **useReducer**, **Context API**, and **component-based architecture** with theme switching and animated counter UI.

---

##  Features

-  Counter using `useReducer`
- Increase count
- Reset count
-  Light / Dark Theme using **Context API**
-  Modern card-based UI with CSS
-  Built with **Vite + React**
-  Animations using **Framer Motion**
-  Optimized with `React.memo`
